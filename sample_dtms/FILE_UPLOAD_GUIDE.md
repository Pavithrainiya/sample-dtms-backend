# File Upload Guide - DTMS

## ✅ Supported File Types

Users can upload **ANY** file type when submitting tasks, including:

### 📦 Compressed Archives
- ✅ `.zip` - ZIP archive
- ✅ `.rar` - RAR archive
- ✅ `.7z` - 7-Zip archive
- ✅ `.tar` - TAR archive
- ✅ `.gz` - GZIP archive

### 📊 Spreadsheets
- ✅ `.xls` - Excel 97-2003
- ✅ `.xlsx` - Excel 2007+
- ✅ `.csv` - Comma-separated values

### 📄 Documents
- ✅ `.doc` - Word 97-2003
- ✅ `.docx` - Word 2007+
- ✅ `.pdf` - PDF documents
- ✅ `.txt` - Plain text
- ✅ `.md` - Markdown

### 📽️ Presentations
- ✅ `.ppt` - PowerPoint 97-2003
- ✅ `.pptx` - PowerPoint 2007+

### 🖼️ Images
- ✅ `.png` - PNG images
- ✅ `.jpg` / `.jpeg` - JPEG images
- ✅ `.gif` - GIF images
- ✅ `.bmp` - Bitmap images
- ✅ `.svg` - SVG vector graphics

### 🎥 Videos
- ✅ `.mp4` - MP4 video
- ✅ `.avi` - AVI video
- ✅ `.mov` - QuickTime video
- ✅ `.webm` - WebM video

### 💻 Code & Data
- ✅ `.json` - JSON files
- ✅ `.xml` - XML files
- ✅ `.html` - HTML files
- ✅ `.css` - CSS files
- ✅ `.js` - JavaScript files
- ✅ `.py` - Python files

## 📤 How to Upload Files

### Step 1: Go to Task Detail Page
1. Login as User
2. Click on any task from your dashboard
3. You'll see the "Submission HQ" section

### Step 2: Select Your File
1. Click the **"Select File to Upload"** button
2. Browse your computer
3. Select any file (ZIP, Excel, PNG, PDF, etc.)
4. File name and size will be displayed

### Step 3: Add Description (Optional)
- Add a message explaining your submission
- Describe what's in the ZIP file
- Provide context for your work

### Step 4: Submit
- Click **"Deploy Submission"** button
- File will be uploaded to the server
- Admin will be able to download and review

## 📦 Best Practices for ZIP Files

### When to Use ZIP Files:
- ✅ Multiple files to submit
- ✅ Project with folders and subfolders
- ✅ Large collections of images
- ✅ Code projects with dependencies
- ✅ Documents with supporting files

### How to Create a ZIP File:

**Windows:**
1. Select all files/folders
2. Right-click → "Send to" → "Compressed (zipped) folder"
3. Name your ZIP file
4. Upload to DTMS

**Mac:**
1. Select all files/folders
2. Right-click → "Compress Items"
3. ZIP file will be created
4. Upload to DTMS

### ZIP File Naming Tips:
- ✅ Use descriptive names: `ProjectName_YourName.zip`
- ✅ Include date: `Assignment1_John_2025-04-03.zip`
- ✅ Avoid spaces: Use underscores or hyphens
- ❌ Don't use: `New folder.zip`, `archive.zip`

---

## 📏 File Size Limits

- **Maximum file size**: 10 MB per file
- **Recommended**: Keep files under 5 MB for faster uploads
- **For large files**: Compress them into ZIP format

### If Your File is Too Large:
1. **Compress it**: Use ZIP, RAR, or 7z
2. **Reduce quality**: For images/videos
3. **Split files**: Upload in multiple submissions
4. **Remove unnecessary files**: Clean up before zipping

## 🔄 Updating Your Submission

You can update your file submission:
1. Go back to the task detail page
2. Select a new file
3. Click "Update Submission"
4. Old file will be replaced with new one

**Note**: Only works if admin hasn't reviewed yet!

## 🚨 Troubleshooting

### "File too large" error
- Compress your file
- Maximum size is 10 MB
- Try reducing image quality or video resolution

### "Upload failed" error
- Check your internet connection
- Try a smaller file
- Refresh the page and try again

### "File not supported" error
- All file types are supported!
- If you see this, it's a bug - contact admin

### Can't see uploaded file
- Refresh the page
- Check if submission was successful
- Look for green success message
