# Gmail Account Diagnostic Checklist

## Step 1: Verify 2-Step Verification

1. Go to: https://myaccount.google.com/signinoptions/two-step-verification
2. Make sure it says "2-Step Verification is ON"
3. If it's OFF, turn it ON and complete the setup

## Step 2: Check App Passwords Page

1. Go to: https://myaccount.google.com/apppasswords
2. You should see a page that says "App passwords"
3. If you see "This setting is not available for your account", it means:
   - 2-Step Verification is not enabled, OR
   - Your account type doesn't support app passwords

## Step 3: Alternative - Enable "Less Secure App Access" (Old Method)

**Note:** Google deprecated this, but try if app passwords don't work:

1. Go to: https://myaccount.google.com/lesssecureapps
2. Turn ON "Allow less secure apps"
3. Try the test again

## Step 4: Check for Account Restrictions

1. Go to: https://myaccount.google.com/security
2. Look for any security alerts or restrictions
3. Check "Recent security activity" for blocked sign-in attempts

## Step 5: Try Alternative Email Provider

If Gmail continues to fail, we can use a free alternative:

### Option A: Use Outlook/Hotmail (Easier)
- Create a free Outlook account
- No app password needed
- Works immediately

### Option B: Use Gmail with OAuth2 (Complex)
- More secure but requires more setup

### Option C: Use SendGrid (Professional)
- Free tier: 100 emails/day
- More reliable for production
